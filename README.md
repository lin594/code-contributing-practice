# code-contributing-practice

> 练习代码贡献中的 `fork` 、`pull request` 等操作。

## 为仓库贡献代码应当怎么做？

- 注意阅读仓库的 `contributing.md` 文档，它通常位于仓库的根目录或 `docs` 文件夹下。
- 认领 issue
- 开始编码
  - 创建 Fork 仓库或同步 Fork 仓库
  - 创建或选定分支
  - 提交代码

- 创建 Pull Request
- 等待代码审查
